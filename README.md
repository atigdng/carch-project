Finding maximum element of array:
Relevant codes: 1.c, 1f.c, int.c for int, float, and initialization respectively.

How to run:
Go to the directory Project.
Run the command 'make clean' to clean all existing files.
Run the command 'make out/int_o3/stats.txt' to run the code 'int'.



Performing matrix multiplication:
Relevant codes: matmul.c, matmul_hybrid.c, mat.c for int, float, and initialization respectively.

How to run:
Go to the directory codes in directory Project.
Run the command 'gcc -O3 matmul.c -o matmul' to compile/execute the file.
Run the command 'time ./matmul' to run the code and see time stats.



Performing dot product of two vectors:
Relevant codes: dot.c, dotfloat.c, dotinit.c for int, float, and initialization respectively.

How to run:
Go to the directory codes in directory Project.
Run the command 'gcc -O3 dot.c -o dot' to compile/execute the file.
Run the command 'time taskset -c 0 ./dot' to run the code and see time stats.
